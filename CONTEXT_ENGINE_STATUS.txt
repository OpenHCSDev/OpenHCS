QUICK START
================================================================================

1. Check if services are running:
   cd /home/ts/code/projects/context-engine
   docker compose ps

2. View logs:
   docker compose logs -f

3. Index your OpenHCS codebase:
   cd /home/ts/code/projects/context-engine
   HOST_INDEX_PATH=/home/ts/code/projects/openhcs \
   COLLECTION_NAME=codebase \
   REPO_NAME=openhcs \
