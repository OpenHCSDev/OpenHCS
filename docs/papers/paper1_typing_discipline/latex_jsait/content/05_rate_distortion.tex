\subsection{Three-Dimensional Tradeoff: Tag Length, Witness Length, Distortion}

Recall from Section 2 that type systems are characterized by three dimensions:
\begin{itemize}
\item \textbf{Tag length} $L$: machine words required to store a type identifier per value
\item \textbf{Witness length} $W$: minimum AST size to implement type identity checking
\item \textbf{Distortion} $D$: worst-case semantic failure flag ($D = 0$ or $D = 1$)
\end{itemize}

\begin{theorem}[Pareto Optimality of Nominal Typing]
Nominal typing achieves the unique Pareto-optimal point in the $(L, W, D)$ space:
\begin{itemize}
\item \textbf{Tag length}: $L = O(1)$ machine words per value
\item \textbf{Witness length}: $W = O(1)$ AST nodes
\item \textbf{Distortion}: $D = 0$ (type equality implies behavior equivalence)
\end{itemize}

Structural typing achieves:
\begin{itemize}
\item \textbf{Tag length}: $L = O(n)$ machine words per value
\item \textbf{Witness length}: $W = O(n)$ AST nodes
\item \textbf{Distortion}: $D = 0$ (type equality implies behavior equivalence)
\end{itemize}

Duck typing achieves:
\begin{itemize}
\item \textbf{Tag length}: $L = 0$ (no explicit tag)
\item \textbf{Witness length}: $W = O(n)$ AST nodes
\item \textbf{Distortion}: $D = 1$ (type equality does not imply behavior equivalence)
\end{itemize}
\end{theorem}

\begin{proof}
See Lean formalization: \texttt{theorems/rate\_distortion.lean}. The proof verifies:
\begin{enumerate}
\item Nominal achieves $(L, W, D) = (O(1), O(1), 0)$ via nominal-tag access
\item Structural requires $O(n)$ bits and $O(n)$ AST nodes to encode structure
\item Duck typing cannot guarantee zero distortion (runtime behavior varies)
\item No other scheme achieves $(O(1), O(1), 0)$
\end{enumerate}
\end{proof}

\subsection{Pareto Frontier}

The three-dimensional frontier shows:
\begin{itemize}
\item Nominal typing dominates all other schemes (minimal on all three dimensions)
\item Structural typing is suboptimal (higher $L$ and $W$, same $D$)
\item Duck typing trades tag length for distortion (zero $L$, but positive $D$)
\end{itemize}

To our knowledge, this is the first formal proof that nominal typing is Pareto-optimal in the $(L, W, D)$ tradeoff for type systems.
